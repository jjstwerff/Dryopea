Loft

A statically-typed scripting language inspired by Rust.

This is the user documentation of the Loft language.

Loft is an imperative language inspired by Rust but designed to be concise and easy to embed.

The pages below are generated directly from the tests/suite/ source files.

Every example can be executed inside the language as it is an integral part of its testing suite.

Key characteristics:

  • Variables are declared on their first assignment; types are inferred.
  • Functions use implicit return: the last expression in a block is the result.
  • Null is a typed absence value; any field or variable can be null unless marked as not null.
  • Strings are UTF-8; len() counts bytes and indexing is byte-based.
  • Collections — vector, sorted, index, hash — share records and can be combined in a single struct.
  • Polymorphic dispatch is supported for struct-enum variants.
